# MarketLoop Mobile

Unified buyer + seller mobile app built with Expo Router, React Native, TypeScript, TanStack Query, Zustand, and NativeWind.

## Scripts

```bash
npm run install:mobile
npm run dev:mobile
npm run typecheck:mobile
```

## Environment

Create `.env` in `apps/mobile/`:

```bash
EXPO_PUBLIC_API_URL=http://localhost:5000/api
EXPO_PUBLIC_SOCKET_URL=http://localhost:5000
```
